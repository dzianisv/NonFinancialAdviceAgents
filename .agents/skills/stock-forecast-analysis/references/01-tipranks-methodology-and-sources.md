# 01 — TipRanks AI Stock Analysis: methodology & data sources (FACT)

Everything here traces to a TipRanks page fetched **2026-07-20**. TipRanks does **not** disclose its
score weights, threshold cut-offs, or LLM prompt — those gaps are flagged explicitly. Do not invent them.

## The two products (do not conflate)

TipRanks serves two different things under `/stocks/{ticker}/...`:

| | Smart Score | **AI Stock Analysis** (this skill's target) |
|---|---|---|
| URL | `/stocks/{ticker}` (overview) | `/stocks/{ticker}/stock-analysis` |
| Score | 1-10 quantitative | **0-100**, LLM-generated |
| Buckets | 8-10 Outperform / 4-7 Neutral / 1-3 Underperform | **>=70 Outperform / 40-69 Neutral / <40 Underperform** (inferred — see below) |
| Method | 8 fixed factors, undisclosed weighting | LLM synthesizes structured data into a report + score + target |
| Price target | none | the AI's **own** 12-mo target, diverges from consensus |

Source: `https://www.tipranks.com/glossary/s/smart-score`, `https://www.tipranks.com/glossary/a/ai-stock-analysis`.

## What the AI report is (FACT, from the glossary + live pages)

Glossary (`/glossary/a/ai-stock-analysis`): *"An AI-powered report that instantly evaluates any stock
using fundamental, technical, and sentiment data. It analyzes **financial statements, earnings call
transcripts, SEC 8-K filings, technical indicators, and more** to calculate an AI-driven score and price
target. The report delivers a concise overview, highlights key strengths and risks, compares peers, and
summarizes corporate events."*

Announcement: `https://www.tipranks.com/news/labs/introducing-stock-ai-analysis-smarter-insights-faster-decisions`.

**Selectable model.** The report header shows a model picker; the score is model-dependent. Live NVDA:
"Outperform **79** (OpenAI-5.2)". So the number is one LLM's opinion, not a deterministic formula — and a
different model yields a different score. The page carries an explicit AI-error disclaimer.

## The exact section stack (FACT — verbatim from the live NVDA report, 2026-07-20)

`https://www.tipranks.com/stocks/nvda/stock-analysis`:

1. **Header** — Rating `79 Outperform` (0-100), model, **AI Price Target `$223.00` (▲10.57% upside)**,
   Action `Reiterated`, Date `07/03/26`.
2. **Summary** — one paragraph explaining the score's drivers and offsets (quoted below).
3. **Positive Factors** — named (e.g. "Product Leadership") + explanation.
4. **Negative Factors** — named (e.g. "Supply Constraints") + explanation.
5. **Key Performance Indicators (KPIs)** — e.g. Revenue by Geography, with "Chart Insights". Data by **The
   Fly** (`pro.thefly.com`).
6. **vs. SPY** + snapshot: Market Cap, Dividend Yield, Avg Volume, P/E, Beta, Revenue Growth, EPS Growth,
   Country, Employees, Sector, **Sector Strength**, Industry.
7. **Share Statistics** — EPS (TTM), Shares Outstanding, avg volumes.
8. **Financial Highlights & Ratios** — PEG, P/B, P/S, P/FCF, EV ratios.
9. **Forecast** — **1Y Price Target `$309.94`**, Price Target Upside, **Rating Consensus `Strong Buy`**,
   **# Analysts `37`**, EPS Forecast (FY) `8.99`, Revenue Forecast (FY) `$393.32B`. → this is the **analyst
   consensus**, shown *separately* from the AI target above (which was $223).
10. **Business Overview & Revenue Model** — Company Description + How the Company Makes Money.
11. **Earnings Call Summary** — call date, **Earnings Call Sentiment `Positive`**, summary, Positive
    Updates, Negative Updates, **Company Guidance** (specific figures, e.g. Q2 rev $91B ±2%).
12. **Financial Statement Overview** — Summary + **sub-scores 0-100**: **Income Statement `98`, Balance
    Sheet `94`, Cash Flow `96`** (each also a Very Positive/Positive/etc. label) + a full financials table
    (TTM + 5 fiscal years: revenue, gross profit, EBITDA, net income, assets, cash, debt, equity, FCF, OCF).
13. **Technical Analysis** — **Sentiment `Neutral`**, Last Price, **Price Trends**: 50DMA `209.81`
    (Negative), 100DMA `198.51` (Positive), 200DMA `192.20` (Positive); **Market Momentum**: MACD `0.15`
    (Negative), RSI `48.22` (Neutral), STOCH `72.47` (Neutral) + a narrative.
14. **Risk Analysis** — "disclosed **24 risk factors** in its most recent earnings report… most in the
    'Tech & Innovation' category", new risks added.
15. **Peer Comparison** — each peer's **Overall Rating (0-100)**: NVDA 79, AMD 78, AVGO 76, TSM 77, QCOM 70,
    INTC 63, Sector 61; + a financial-indicators table (Mkt Cap, P/E, ROE, Div Yld, Rev Growth, EPS Growth)
    + performance comparison.
16. **Corporate Events** — dated, sentiment-tagged 8-K-derived events.
17. **Disclaimer** — "automatically generated by our AI systems… may contain inaccuracies… not financial
    advice." + Date of analysis.

**Score threshold inference (the buckets are NOT published).** From the peer table: QCOM `70` = Outperform,
INTC `63` = Neutral, Sector `61` = Neutral. So the Outperform cut is between 63 and 70 — we use **>=70
Outperform, 40-69 Neutral, <40 Underperform**, and we state it is inferred, not disclosed.

**AI target ≠ consensus (the defining feature).** Live gaps observed 2026-07-20:
- NVDA: AI target **$223** vs analyst consensus **$309.94** (−28%).
- AAPL: AI target **$387** vs analyst consensus **$329.44** (+17%).
So the AI product is a **separate opinion engine**, not a repackaging of the Street. Reproducing it means
building an independent target and showing it beside the consensus.

Verbatim NVDA summary (score rationale — tells us the visible weighting):
> "NVDA scores highly due to **outstanding financial performance (profitability, cash flow, and balance
> sheet strength)** and an **earnings call** that reinforced strong guidance, demand visibility, and
> aggressive shareholder returns. The main offset is **weak near-term technical momentum** (below key
> moving averages with negative MACD), with **valuation remaining premium** and a very low dividend yield
> providing limited support."

→ Reading the drivers straight off the page: **financials (heavy) > earnings call > technicals (offset) >
valuation (offset) > dividend (minor).** That is the basis for our section weights in `references/02`.

## Public data sources to reconstruct each section (our analysis)

| Section | Free/cheap source | Reliability |
|---|---|---|
| Financial Statement Overview + sub-scores | **SEC EDGAR XBRL `companyfacts`** (`https://data.sec.gov/api/xbrl/companyfacts/CIK{cik}.json`), `yfinance` financials, StockAnalysis.com, Finviz | High — enables Piotroski, margins, accruals, FCF |
| Technical Analysis (MAs/MACD/RSI/STOCH) | price via `yfinance`/Stooq → `analyse-technical`'s `scripts/ta.py` (`pandas`/`ta`) | Fully reproducible |
| Valuation & KPIs | `yfinance` `.info`, StockAnalysis.com, Finviz, Macrotrends | High |
| Analyst Forecast (consensus, target, # analysts, EPS/Rev FY) | `analyse-sellside` (Yahoo/StockAnalysis/TipRanks/MarketBeat), Finnhub, FMP | Medium-high for consensus level |
| Earnings Call Summary + guidance | earnings-call transcript (`web_fetch` of Motley Fool/Seeking Alpha/company IR), `hedgefund-jpmorgan-earnings` | Transcript often free; sentiment reproducible |
| Risk Analysis | latest **10-K/10-Q "Risk Factors"** (EDGAR full-text) | High (official) |
| Corporate Events | **EDGAR 8-K** filing feed (`https://www.sec.gov/cgi-bin/browse-edgar?action=getcompany&type=8-K`), `read-news` | High (official) |
| Peer Comparison | run this skill's scoring across named peers; peer list from Finviz/StockAnalysis sector | Reproducible |
| Business Overview | `yfinance` `longBusinessSummary`, company 10-K Item 1, StockAnalysis profile | High |
| KPIs (segment/geography revenue) | 10-K/10-Q segment notes (EDGAR), The Fly (paywalled) | Medium — segment data needs filing parse |
